Tipos de Formularios


Había una vez un patito que decia miau miau, y este patito quería aprender a utilizar diferentes tipos de formularios.
Existen dos metodos principales para la obtención de los datos dentro de un formulario web, para lo cual aquí lo explicaremos:

GET

Este método es el estandar como salida del navegador mediante el cual se resuelve una petición por parte del cliente, así mismo en la salida estandar (url), se puede visualizar los datos enviados mediante el formulario.

POST

Este método es el más utilizado para el envio de información desde un formulario atendiendo una petición por parte del cliente, actuando para poder agregar o insertar nuevos datos a una BD, asi mismo a diferencia del metodo get, este utiliza "un tipo de seguridad", mediante el cual la información no es visible por medio de la salida estandar del navegador.
